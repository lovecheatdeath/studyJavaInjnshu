1.背景介绍

随着响应式设计模型的诞生，Web网站发生翻天腹地的改革浪潮，下面就来讲一下实现响应式设计至关重要的一个知识点，媒体查询。

2.知识剖析

媒体查询可以让我们根据设备显示器的特性（如视口宽度、屏幕比例、设备方向：横向或纵向）为其设定CSS样式， 媒体查询由媒体类型和一个或多个检测媒体特性的条件表达式组成。媒体查询中可用于检测的媒体特性有 width 、 height 和 color （等）。使用媒体查询， 可以在不改变页面内容的情况下，为特定的一些输出设备定制显示效果

3.常见问题

问题：在CSS中媒体查询用到的PX究竟是怎样一个单位？它是相对单位还是绝对单位？

4.解决方案

问题：css里所有的px都是css pixel（ 同时media query使用的px显然也是css pixel。），其显示大小是相对的而不是绝对的，是设备相关的。 而css pixel * devicePixelRatio = 实际屏幕像素。

5.编码实战

                
                    以960px为分界线，改变background颜色，媒体查询的实际应用。
                    等于960px尺寸的代码：
                    @media screen and (max-device-width:960px)
                    {background:red;}
                    }

                    然后就是当浏览器尺寸小于960px时候的代码了：
                    @media screen and (min-width:960px){
                    body{background:orange;}
                    }


                    我们还可以混合使用上面的用法：
                    @media screen and (min-width:960px) and (max-width:1200px){
                    body{background:yellow;}
                    }
                    上面的这段代码的意思是当页面宽度大于960px小于1200px的时候执行下面的CSS。
