1.背景介绍

javascript是一门面向对象的弱类型语言，其所有数据类型都可以称之为对象。面向对象就是使用对象。面向对象开发就是使用对象开发。

对象和数组都是JS学习中经常用到的数据类型，学习理解他们的异同点显得尤为重要

2.知识剖析

对象和数组都可以表示数据的集合

数组：数组是数据的有序集合。每个数据值叫做一个元素，而每个元素在数组中有一个位置，以数字表示，称为索引或下标（由0开始）。并且包含一个名为length的属性，该属性表示数组元素的个数！

数组中值, 可以为任何类型. 数组, 对象, null, undefined.

创建数组：

var arr = [];  var arr = new Array()

读写数组内数据使用：数组名+[索引值]的方式;

                       
                       var a = [1,2,3];
                       a[1] //2;
                       a.length //3
                       a[0]="test";
                       a //["test","2","3"]
                   
                   
                        
设置数组长度

                        
                       var a=[]；
                       a.length=5;
                       a[1]="元素"；
                       a//[undefined × 1, "元素1", undefined × 3]

一般不去设置数组length属性

数组元素添加

push()方法可以将一个或多个新元素附加到数组的尾部, 然后返回数组新长度

pop()删除数组中的最后一个元素, 减少数组的长度, 返回它删除的值

直接对原数组操作

var a = [1,2,3,4]; a.push("2");//5 a//[1,2,3,4,"5"] a.pop();//"5" a//[1,2,3,4]

delete 运算符把一个数组元素设置为 undefined 值, 但是元素本身还是存在的

其他删除方式：Array.shift();【删除第一个】 Array.splice();【从一个数组中删除一个连续范围】；slice()截取元素，不改变数组

                              
                           var a = [1, 2, 3]; delete a[1]; 
                           console.log(a); // [1, undefined, 3];

遍历数组：

var array = [1, 2, 3, 4, 5];  for (var i = 0, l = array.length; i < l; i++) {      console.log(array[i]); }//依次输出：1 2 3 4 5

数组还有其他方法：join()返回用字符连接的字符串,sort()排序等

对象： 是一个无序属性集合, 每个属性都有自己的名字和值，无length属性，访问对象属性值：对象.属性名（对象.方法（））or对象[“属性名”]。

注： []里面使用属性名(可使用变量).

创建对象：

var obj = {};  var obj = {name: 'maxthon'}; var obj = {name: {}, text: []}

要获得对象的属性数，需用遍历方法

var a={name:"xiaoming",a:"b",3:"c"},i=0; for(var x in a){ i++ } console.log(i)//3 a.name//"xiaoming"

/检查属性存在性;in

var a={name:"xiaoming",a:"b",3:"c"};"name" in a //true

删除属性：用操作符delete

a={name:"xiaoming",a:"b",t:"c"};delete a.a //true a//{name:"xiaoming",t:"c"}

3.常见问题

在操作一个对象或数组的时候，想取出某个值为X的数，相同的字面量却无法执行判断或循环内的函数

var a={name:"xiaoming",a:"b",b:"c",test:"5"}, b=[1,3,4,5]; if(a.test===b.3){//false ... }

4.解决方案

实际使用中，可能会忽略对象的类型判断

可以console.log(typeof(a.test)) console.log(typeof(b.3))

string number 两种不同的对象，返回false

1.parseInt(a.test)转换为number类型或toString(b.3)

2."==="表示类型和值都完全相等，换成"=="
