1.背景介绍

HTML页面中元素的大小、位置、形状等布局样式都需要通过一种或者多种长度单位进行调整控制，以达到设计要求的样子。

一般情况下，我们常常见到和使用px、em、%等单位。

随着html、css规范的发展以及一些显示屏幕技术的突破，手机使用率的增长，一些新的长度单位也被定义并应用起来，例如vw、vh、vm等。

2.知识剖析

2.1CSS长度单位

CSS长度单位可粗略按绝对长度单位和相对长度单位两大类进行区分。

绝对长度单位：px、pt(point)、pc(Picas派卡)等

相对长度单位：em、rem、vw、vh、vm、%等

2.2CSS长度单位概念简介

px：即像素，计算机屏幕上最小的单元。显示器分辨率越高，则所含的像素越多。

pc、pt使用不是很多，在这里就不做过多介绍了。

em：相对于当前对象内文本字体的大小，如果当前文本没有设置字体大小，则按相对于浏览器的默认字体大小。

rem：相对于根元素（html元素）的字体的大小。

%：相对于具有相同属性父元素的宽度或字体大小。

vw：等于视窗宽度的1/100。例如浏览器的宽是900px,则1vw的值为9px。

相对应的，vh：等于视窗高度的1/100。例如浏览器的高是640px,则1vh的值为6.4px。

vm：一般写成vmin，是相对于以上vw、vh而言的，是两个值的较小的那个。
