一、背景介绍

1.nginx是什么

Nginx是一个高性能网页服务器

那么什么是高性能网页服务器呢？

不百度，不谷歌来思考一下这个问题，对于普通用户群体来说，什么是一个高性能站点

一般的用户所反能接受的打开网页的速度是在4秒以内越快越好，很有可能超过4秒你的网页可能就会被别人所关闭，那么高性能所指的，就是指哪些响应速度很快的哪些网站。今天我给大家介绍的Nginx就是能帮你构建出这种高性能服务器的一个平台

二、知识剖析

1.谁选择了Nginx

国内

百度、京东、新浪、网易、腾讯、知乎、搜狐、盛大

淘宝使用的是自己改造过的nginx，Tengine

国外

比较出名的有Wordpress、Instagram、Facebook、GitHub

nginx有什么优点

  1. 支持SLB
  2. 更加高效，更加安全
  3. 快速传输媒体
  4. 支持WEB加速提升移动端性能
  5. 防DDos攻击
  6. 跨平台
  7. 高并发
  8. 优点还有很多，比如反向代理和正向代理

三、怎么安装

如何安装nginx

  1. 选择你对应的系统版本
  2. Windows系统可以到官方网站http://nginx.org/下载对应的版本
  3. 如果你是Linux系统可以使用具体的命令行进行操作

    4. 如果是Centos系统
    sudo yum install nginx
    如果你是Ubuntu系统可以使用
    sudo apt install nginx

  4. MAC系统也可以通过命令行的方式来进行安装

    5. 先brew search nginx搜索nginx
    使用brew install nginx进行安装

四、实际操作

Ubuntu

  1. 使用putty、Bitvise等工具连上你的服务器
  2. 使用root账户
  3. 输入命令行 Sudo apt ingstall nginx
  4. 把你的网页上传到服务器上,可以通过各种方式FTP等
