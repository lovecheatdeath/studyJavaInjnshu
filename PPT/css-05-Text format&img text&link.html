2.文本格式

title与h1的区别

定义：title是网站标题，h1是文章主题

作用：title概括网站信息，可以直接告诉搜索引擎和用户这个网站是关于什么主题和内容的；h1突出文章主题，面对用户，更突出其视觉效果，指向页面主体信息

b与strong、i与em的区别

定义：b与i是文字显示样式，而strong与em表示文本为强调内容

规范：em用于强调文本，strong的强调程度更强一些。b与i只表示外观，应尽量少用

3.图像的文本

img中的alt与title属性

title是鼠标滑过时显示的文字提示，规定图像的额外信息

alt属性用来为图像定义一串预备的可替换的文本，在图像无法显示时显示替换文本

4.链接属性

URL属性

绝对 URL - 指向其他站点

                
   
   example

相对 URL - 指向站点内的文件

src与href的区别

定义：href指定网络资源的位置建立链接或关系，用在link和a等元素上。src将外部资源嵌入到当前标签所在位置，如img图片和js脚本等

加载：浏览器解析src属性时，会暂停其他资源的下载和处理，直至将该资源加载，编译，执行完毕。浏览器解析到href的时候会识别该链接内容，对其进行下载不会停止对当前文档的处理
