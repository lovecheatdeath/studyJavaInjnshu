1.背景介绍

CSS3可以实现很多漂亮的图形，直接用CSS3画出这些图形，要比贴图性能更好，体验更加，是一种非常好的网页美观方式。

2.知识剖析

在css绘制图形时，一般要使用border-radius,transform属性。

2.1利用border-radius属性绘制椭圆 
                   
                     div{
                     width:120px;
                     height:90px;
                     margin:100px;
                     background-color:green;
                     border-radius:50%;
                     }
2.2利用border属性绘制半个椭圆 
                   
                    border-radius:50%/100% 100% 0 0;
2.3利用border属性绘制4分之1个椭圆 
                   
                     border-radius:100% 0 0;
2.4利用border属性绘制其他椭圆 
                   
                       border-radius:66% 33%;
                       border-radius:50%/80% 20%;
                       border-radius:50%/66% 66% 33% 33%;
                       border:50% 0;
                       border-radius:100% 0;
2.5使用rotate旋转两个正方形 
                   
                    div::after {
  content:"";
  position:absolute;
  top:0;
  left:0;
  right:0;
  bottom:0;
  background-color:green;
  transform:rotate(45deg);
}
div{
  position:relative;
  width:150px;
  height:150px;
  background-color:green;
  margin:100px;
}
