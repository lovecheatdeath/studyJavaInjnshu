1.背景介绍

早在CSS2.1的时候就引入了content属性，应该算是最常用的一个css属性之一， 比如我们创建一个聊天气泡、超链接的立体翻转特效、添加图标。都要使用到content属性， 当然这个属性要结合：before和：after伪类来使用。下面详细介绍它的用法。

2.知识剖析

content属性的作用是，配合 :before 及 :after 伪元素使用，来插入生成内容。

首先来看一下content的属性值有哪些及用法

1.插入纯文字
content:"直接插入的文章"，或者content:none不插入内容

                
                    
这是h1

                    
这是h2

CSS

                
                   h1::after{
                   content:"h1后插入内容"
                   }
                   h2::after{
                   content:none
                   }

2.嵌入文字符号
可以使用content属性的open-quote属性值和close-quote 属性值在字符串两边添加诸如括号、单引号、双引号之类的嵌套文字符号。 open-quote用于添加开始的文字符号，close-quote用于添加结束的文字符号

                
                h1{
                quotes:"(" ")";  /*利用元素的quotes属性指定文字符号*/
                }
                h1::before{
                content:open-quote;
                }
                h1::after{
                content:close-quote;
                }
                h2{
                quotes:"\"" "\"";  /*添加双引号要转义*/
                }
                h2::before{
                content:open-quote;
                }
                h2::after{
                content:close-quote;
                }
