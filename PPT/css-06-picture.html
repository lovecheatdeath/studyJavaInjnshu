1.背景介绍

css绘图，或者说用css属性在页面中呈现出一些特殊的图案效果，例如下拉菜单中的倒三角，页面转换时页面中下部的圆圈等，如何用纯css来实现

2.知识剖析

css绘制特殊图案并不是通过很多的div拼凑起来，而是通过修改div内的属性来实现，如background-color，border-radius等属性

3.常见问题

为什么要用css画图，直接切图不是更方便快捷么？css绘图的优点和缺点有什么？

4.解决方案

为什么要用css画图，直接切图不是更方便快捷么？

简单的图形在页面中很多，用css可以减少http的请求

css呈现的是矢量图，缩放不会失真

不用为了一个小图标的缩放去跪着求UI！！

css绘图的优点和缺点有什么？

减少http请求，页面加载速度提升

无法呈现出复杂图形

遇到复杂图形还是得求UI妈妈！！

5.编码实战

圆：有几点需要注意，其一宽度和高度值相同，其二圆角值为宽度或高度值的一半。

                
                    #circle {
                    width: 100px;
                    height: 100px;
                    background: red;
                    -moz-border-radius: 50px;
                    -webkit-border-radius: 50px;
                    border-radius: 50px;
                    }

半圆：制作半圆和圆使用的方法是一样的，但需要配合元素的高度，宽度以及圆角的方位，制作出半圆形效果。border-radius:↖ ↗ ↘ ↙ 

                
                    #semicircle{
                    width: 100px;
                    height: 50px;
                    background: red;
                    -moz-border-radius: 100px 100px 0 0;
                    -webkit-border-radius:  100px 100px 0 0;
                    border-radius:  100px 100px 0 0;
}

三角：最常用的小图标 

                
                    #triangle{
                        width: 0;
                        height: 0;
                        border-top: 40px solid green;
                        border-right: 40px solid blue;
                        border-left: 40px solid black;
                        border-bottom: 40px solid red;
                    }

三角：实例一个方向，其他方向只需要改变transparent所在的位置即可 

                
                    #triangle-right{
                        width: 0;
                        height: 0;
                        border: 50px solid;
                        border-color: transparent transparent transparent red;
                    }

汉堡图标：修真院任务中常见的小图片，用

娱乐 吃豆人：仿三角形做法，用width和height为0来做一个圆，然后将一个border方向设置为transparent透明 

                
                    #Pac-man{
                        width: 0;
                         height: 0;
                        border: 60px solid;
                         border-color: red transparent red red ;
                        -moz-border-radius: 60px;
                        -webkit-border-radius: 60px;
                        border-radius: 60px;
                    }
