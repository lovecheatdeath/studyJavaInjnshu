js基础类型和常用方法

    js的基础数据类型分三种
  1. 数字,数字也包括了浮点型(小数)等number
  2. 字符串string
  3. 数组array
  4. 布尔值boolean
  5. null
  6. undefined

js是一门弱类型语言,声明变量时可以不用标明其类型,变量在做一些操作是会隐式的转换类型,虽然很方便但是也有一些问题,每一个基础类型储存的地址都是固定不与其他变量共享的,并且是按值来传递,而对象类型储存的是一个引用地址,值得传递是引用传递,这样就会存在像下面这样的情况

var s ={ a :1, } var b = s ; b.a=2; console.log(s.a); //2

用typeof可以识别出基本类型的具体类型

如: typeof 2 // "number"

一、数字类型的方法

  • Math.abs() 绝对值
  • Math.round() 四舍五入
  • Math.ceil() 向上取整
  • Math.floor() 向下取整
  • Math.max() 取最大数
  • Math.min() 取最小数
  • Math.random() 取大于等于0小于1的数
  • parseInt() 转字符串为数字,忽略第一次不能识别和后面所有的字符
  • parseFloat() 转字符串为浮点,忽略第一次不能识别和后面所有的字符
  • toFixed() 保留小数点位数,输出的类型为字符串
  • Number() 转数字只要有无法识别的字符就返回NaN
    1. 浮点数进行计算时会产生输出的结果有误差,
      • console.log( 1 - 0.8 ); //输出 0.19999999999999996
      • console.log( 6 * 0.7 ); //输出 4.199999999999999
      • console.log( 1.2 / 0.2 ); //输出 5.999999999999999
      • console.log( 0.1 + 0.7 ); //输出 0.7999999999999999

    2. 用(!isNaN())的方法来判断是否为数字

    二、字符串

    1. charAt() 输入字符返回索引
    2. indexOf() 输入字符返回索引,无则返回-1
    3. search() 输入指定的值或正则表达式,如果匹配不到就返回-1,存在则返货索引位置
      'abc'.search(/[b]/) //1
      'abc'.search('b') //1
    4. match() 输入指定的值或正则表达式,如果匹配不到就返回null,匹配到则以数组返回
      'abc'.match('b') //["b"]
      'abc'.match(/[b]/) //["b"]
    1. replace() 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。
      stringObject.replace(传入正则或具体的字符,替换的字符或者函数)
      如果传入函数,替换的字符就为函数返货出来的字符
    2. substring(下标a,下标b) 用下标查找a开始-b结束但是不包括b本身,如果b不传就去从a开始到末尾的所有字符
    3. slice(下标a,下标b) a规定从何处开始选取。如果是负数，那么它规定从数组尾部开始算起的位置。也就是说，-1 指最后一个元素，-2 指倒数第二个元素，以此类推,b是结束位置,如果这个参数是负数，那么它规定的是从数组尾部开始算起的元素。
    4. substr(下标a,n个数) 从a开始取n个数.
    1. split() 传入字符或者正则,已传入的参数分割字符串,返回一个数组
    2. toLowerCase() toUpperCase() 输出大写或小写字符
    3. 转义字符
    4. String() 转字符串
    5. Boolean() 转布尔值
