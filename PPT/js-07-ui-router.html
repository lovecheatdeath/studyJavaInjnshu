2.知识剖析

首先在angular之中实现路由跳转功能有其自带的ngRoute还有常用的第三方插件ui-router

选择ui-router来替代自带的ngRouter是因为它更好用更灵活，具体对比则不是本次分享内容

想要页面之间传递参数，则首先要实现跳转
使用ui-router实现跳转有两种方式

一种是使用其ui-sref指令，另一种是使用$state.go()方法

$state.go()方法

通常用在controller里面，如

                    
                    .controller('heiheihei', function($scope, $state) {
                    $state.go('login');
                    });
